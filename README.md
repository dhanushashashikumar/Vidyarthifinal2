# 🚍 Vidyarthi Bus

A smart student transportation app for real-time bus tracking, crowd monitoring, and campus mobility.

## 📱 Features
- Live Bus Tracking using GPS
- Crowd Status Indicator (Green/Yellow/Red)
- Next Bus Timings
- Student Login System
- Bus Reporting System
- Notices & Alerts
- Student Community Chat
- Shared Auto/Ride System

## 🛠️ Tech Stack
- Android Studio (Java/Kotlin)
- XML for UI design
- Firebase Realtime Database
- Firebase Authentication
- Firebase Cloud Messaging
- Google Maps API
- GPS Services
